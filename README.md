# NetAssist

🌐 网络诊断与监控桌面工具，基于 Tauri + React 构建。

## 功能

- 网络连通性检测（Ping / TCP / HTTP）
- 实时网络状态监控面板
- DNS 查询与解析
- 端口扫描
- 网络延迟统计图表

## 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **后端**: Rust + Tauri
- **构建**: Vite

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建发布
npm run tauri build
```
