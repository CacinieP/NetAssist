## 断网急救箱

### 故障诊断与修复

- **流量异常**
    
    → 显示占用排行 → 关闭高耗应用 → QoS限速
    
- **IP/连接失败**
    
    → 检测双栈可用性 → 切换优先级 → 重置适配器 → 重获IP
    
- **DNS故障**
    
    → 测试响应 → 切换备用DNS（8.8.8.8/1.1.1.1）→ 清缓存 → 直连IP
    
- **延迟/丢包**
    
    → 显示指标 → 诊断路由 → 建议切换网络（热点）→ MTU优化
    

### 核心功能

1. **一键诊断**：分层扫描（30秒完成）
2. **智能决策**：推荐2-3个修复操作
3. **快捷修复**：重置网络栈/切换DNS/断开异常连接/切换IPv6
4. **离线知识库**：内置故障码与解决方案
5. **日志导出**：诊断数据保存

### 技术要点

- **架构：**守护进程 + 轻量GUI（Electron/Qt）
- **权限：**管理员权限
- **兼容：**跨平台API适配（Windows/Linux/macOS）
- **体验：**红绿灯状态 + 进度显示 + 撤销功能
